# Relation-Attention Semantic-Correlative Knowledge Graph Embedding for Inductive Link Prediction

RASC is implemented based on Pytorch.

## Install 

Make sure your local environment has the following installed:
```
    Python == 3.6.5 
    torch == 1.4.0 
    numpy == 1.16.5 
    scipy == 1.3.1 
    sklearn == 0.21.3
```

## Dataset files

 `data/`
  - `FB15k/`
  - `FB15k-237/`
  - `NELL995/`
  - `DDB14/`
  - 'WN18/'
  - 'WN18RR'

## Training

```
$ python main.py --cuda
